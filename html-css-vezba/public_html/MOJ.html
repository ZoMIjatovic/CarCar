1.1 IP Adresa Svaki računar na Internetu poseduje IP adresu koju obezbeđuje ISP i koja ga jednoznačno određuje. Da bi komunicirali sa drugim računarom na Internetu neophodno je da znate njegovu IP adresu

IP adresa se predstavlja sa 4 broja u opsegu od 0 do 255 razdvojenim tačkom, na primer: 194.9.94.86

2.3 Web server Web server je računar na Internetu na kome se fizički nalazi web sadržaj. Web browser komunicira sa web serverom i od njega dobija traženi sadržaj. Na jednom web serveru može se nalaziti sadržaj od više web sajtova tj. jedan web server može da "hostuje" više domena. 2.4 HTTP HTTP je protokol po kome komuniciraju web browser (klijentski program) i web server (serverski program).
